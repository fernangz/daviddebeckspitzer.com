# David de Beck Spitzer - Portfolio Website

Personal portfolio website for David de Beck Spitzer, a graphic designer and art director.

## Structure

- `index.html` - Main HTML file
- `image/` - Favicon files
- `images/` - Gallery images and assets (logo, signature)
- `font/` - Custom fonts (Outfit)
- `data/` - Web app manifest
